# Instructions pour ajouter la police Uber Move

## Étape 1 : Placer les fichiers de police

Place tes fichiers de police Uber Move (format .otf) dans le dossier `public/fonts/` :

```
FADY LANDING PAGE/
  ├── public/
  │   ├── fonts/
  │   │   ├── UberMove-Medium.otf
  │   │   └── UberMove-Bold.otf
  │   ├── logo.png
  │   └── ...
```

**Important :** Les fichiers doivent s'appeler exactement :
- `UberMove-Medium.otf` (pour le poids Medium - 500)
- `UberMove-Bold.otf` (pour le poids Bold - 700)

## Étape 2 : Vérification

Une fois les fichiers en place, la police Uber Move sera automatiquement appliquée à tout le site avec :
- **Medium (500)** pour le texte normal
- **Bold (700)** pour les titres et éléments en gras

## Note

Si tes fichiers ont des noms différents, dis-moi les noms exacts et je les mettrai à jour dans la configuration.
